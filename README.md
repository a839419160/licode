# Licode

## 1、只支持centos (only support centos)
* 线上系统基本都用centos
* 没有时间适配其他系统

## 2、示例 (usage)

	cd scripts
	
	#install deps
	./installCentosDeps.sh
	
	#install erizo
	./installErizo.sh
	
	#install nuve
	./installNuve.sh
	
	#install example
	./installBasicExample.sh
	
	#start licode
	./startLicode.sh
	
	#start basic example
	./startBasicExample.sh
	
	#stop licode
	./stopLicode.sh
	
	#stop basic example
	./stopBasicExample.sh
	
	#restart licode and basic example  ^_^
	./restartAll.sh
